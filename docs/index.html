Implementation & Usage

  • Download the latest skewuy.min.js from this Github repo and place it in your project directory.
  • Add the following tag within head or body of your index.html This reference should be placed before the reference to the main javascript of your page.

    <body>

    ...

    <script src = "./skewuy.min.js"></script>

    <script src = "main.js"></script>

    </body>

  • Create a container for Skewuy Element in your HTML document.

    <body>

    ...

    <div id = "skewuyContainer"></div>

    ...

  • You need a spritesheet that contains the ordered sequence of images. The dimension of each image in the spritesheet is currently set to 1920 x 1080 px
  • On your main.js define the following to use as Skewuy properties.

    // Height of the Skewuy Element

    // The height should be at least 200 for better experience

    const skHeight = 200;


    // Number of frames in the spritesheet

    const skframeCount = 15;


    // Gap from left of the spritesheet

    const skframeGutter = 86;


    // URL of the spritesheet image

    const spriteImg = "./res/sprite.jpg";


    // Skewuy Container Element created previously

    const skContainer = document.getElementById("skewuyContainer");

  • Instantiate Skewuy and pass the above declared params.

    const skewuy1 = new Skewuy(skHeight, skframeCount, skframeGutter, spriteImg, skContainer);

  • And you're done!
